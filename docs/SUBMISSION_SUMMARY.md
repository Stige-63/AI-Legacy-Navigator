# Submission Summary

AI Legacy Navigator is a youth-centered guided planning application built by the Youth Legacy
Initiative during Build Week. It helps young people in grades 6–12 reflect on their strengths,
interests, values, and goals, then turns that reflection into practical next steps. Users complete
a brief, judgment-free questionnaire — no accounts, no sensitive personal data — and instantly
receive a personalized Legacy Profile, three realistic pathways (education, career/
entrepreneurship, and community service/leadership), and a simple 30-day action plan broken into
Discover, Learn, Practice, and Share-or-Serve. Results can be copied or printed, and users can
start over at any time.

The app runs entirely in a reliable Demo Mode, generating encouraging, personalized results from
the user's own answers with no external dependencies — with a clean, optional integration point to
connect live AI generation through a protected serverless endpoint if desired. Built with React
and Vite as a lightweight single-page application, it required no database and stores nothing
between sessions. Responsible-AI guardrails are built in throughout: no diagnosis, no guaranteed
outcomes, no collection of names, addresses, or school information, and a clear educational-use
disclaimer on every results screen encouraging youth to involve a trusted adult in major decisions.

AI Legacy Navigator is a working, mobile-responsive MVP ready to demonstrate end-to-end today.
